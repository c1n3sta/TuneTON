# UI Components Workflow

## Component Structure
- `PlayerControls`: Play/pause, skip, progress bar
- `EffectControls`: Sliders for pitch, tempo, EQ bands
- `TrackInfo`: Displays current track metadata
- `ThemeProvider`: Handles Telegram theme integration

## Development Rules
1. Use CSS Modules for scoped styling
2. Follow Telegram's design guidelines
3. Make all components responsive
4. Implement proper ARIA labels
5. Use React.memo for performance optimization
