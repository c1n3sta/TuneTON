# TuneTON Monitoring Suite

This directory contains all monitoring tools and configurations for the TuneTON application.

## Components

1. **PM2 Process Monitoring** - For Node.js process management and monitoring
2. **Uptime-Kuma** - For uptime monitoring of services
3. **Prometheus + Grafana** - For comprehensive metrics and dashboards
4. **ELK Stack** - For log aggregation and analysis

## Directory Structure

- `/pm2` - PM2 configuration and monitoring
- `/uptime-kuma` - Uptime-Kuma configuration
- `/prometheus-grafana` - Prometheus and Grafana configurations
- `/elk-stack` - ELK stack configurations
- `/dashboards` - All-in-one monitoring dashboard

## Setup Instructions

Each component has its own setup instructions in its respective directory.
